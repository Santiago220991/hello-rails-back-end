# Hello Rails Back-End

> This is is a project to learn how add react-redux to a rails app.


## Link to Front-End-App

https://github.com/Santiago220991/hello-react-front-end


## Built With

- Rails, Ruby.

## Setup Project
- Clone repo by using (https://github.com/Santiago220991/hello-rails-back-end.git)
- cd hello-rails-back-end
- Install dependencies with `bundle install`

## Usage
- Run  `rake db:create`
- Run `rails server`
